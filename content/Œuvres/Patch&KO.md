# Patch&KO

## Description

Avec Patch&KO, Antonin Fourneau et Manuel Braun injectent une dose de hasard dans le jeu Street Fighter 2, la manette étant remplacée par une bille à trajectoire chaotique, comme dans le Pachinko. Pour gagner, il faut accepter de perdre le contrôle.

« Pour gagner, il faut accepter de perdre le contrôle. » — Marie Lechner (Libération, 19 janvier 2008)

Créé avec Manuel Braun.
