# Ikeastation

## Description

Ikeastation est une table « Lack » IKEA transformée en jeu. La table devient un substitut à l'interface-écran couramment utilisée dans un jeu vidéo. Jouer avec le joystick provoque des vibrations qui guident les mouvements d'un verre sur la table.

Un Ouija est l'un des rares moments d'interaction ludique avec le mobilier. Ikeastation est une nouvelle invitation à jouer, mais avec les meubles.
