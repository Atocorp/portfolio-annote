# PcMotoTuning

## Description

PcMotoTuning joue sur la rencontre improbable entre deux pratiques de tuning : la modification d'ordinateurs et les vélos mexicains « lowrider » des années 1980. Avec une touche de dérision, le public doit réellement s'asseoir sur l'ordinateur pour lancer un jeu, puis tourner le guidon pour déplacer l'avatar.

Créé avec Erational.
