# PongNet

## Description

Deux Gameboys connectées par un câble sont posées sur une réplique miniature d'une table de ping-pong. Les visiteurs jouent à une version classique du jeu vidéo « Pong ». Contrairement au jeu original, la balle n'est pas visible par les deux joueurs en même temps : elle passe d'un écran à l'autre, créant une surprise.

L'œuvre est une métaphore d'Internet dans lequel un paquet de données — la balle — s'échange d'un terminal à l'autre à travers un réseau.

Créé avec Erational.
