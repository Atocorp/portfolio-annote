# Ghostpad

## Description

« Ghostpad » est un joystick qui joue tout seul. Le joystick se déplace dans toutes les directions à des vitesses variables. Les mouvements sont parfois rapides et furieux. Ces gestes évoquent nos souvenirs de jeux : on peut imaginer, malgré l'absence d'écran, le jeu qui se déroule devant nos yeux. Cette pièce interroge notre mémoire collective.
