# Moripad

## Description

Moripad est une interface de jeu fantasmée : quel jeu utiliserait cette interface ? C'est un objet spéculatif qui n'est pas seulement un outil de jeu, mais quelque chose que l'on désire comme un bijou derrière une vitre. On ne peut pas interagir avec le Moripad, on peut seulement l'admirer.

Cette création est un hommage à l'œuvre « Prière de toucher » de Marcel Duchamp. (* mori = forêt en japonais)
