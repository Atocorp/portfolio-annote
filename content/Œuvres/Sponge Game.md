# Sponge Game

## Description

Sponge Game est une installation qui demande aux gens de se toucher pour interagir dans un jeu normalement solitaire. Toucher la main d'une personne permet de se déplacer à droite, une autre de tourner. Chaque personne est une fonction du jeu et le groupe doit s'organiser et collaborer pour jouer. La mécanique du jeu, le gameplay, devient une sorte de performance entre des personnes qui ne se connaissent pas mais se touchent.

Créé avec Manuel Braun.
