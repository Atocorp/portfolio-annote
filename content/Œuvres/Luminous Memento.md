# Luminous Memento

## Description

Luminous Memento est un mémorial en hommage aux citoyens de Poitiers morts dans divers conflits depuis la Première Guerre mondiale. Les noms des défunts apparaissent et traversent un béton translucide, gravant numériquement leur mémoire dans la pierre. À chaque anniversaire de leur naissance, leur nom revient plusieurs fois pour commémorer, au-delà de leur mort, la vie de ces citoyens qui ont défendu la France.

La base de données comptait 2107 noms à l'inauguration le 11 novembre 2016. Un site web (luminousmemento.fr) permet de consulter la date de passage du nom d'un défunt sur le mémorial.

Programmation : Manuel Deneu. Électronique : Alexandre Saunier et Grégoire Lauvin. Design graphique : Loïc Horellou.
